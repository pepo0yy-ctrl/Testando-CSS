## 🎬 Meu Portal de Entretenimento Digital
Um projeto simples em HTML e CSS que organiza conteúdos de Filmes e Séries, Música e Podcasts e Games e Tecnologia em um layout responsivo e estilizado.

## 🛠️ Tecnologias Utilizadas

HTML5: Estrutura semântica da página.
CSS Reset (Eric Meyer): Remove estilos padrão dos navegadores.
CSS externo (style.css): Estilos principais do portal.
CSS interno (<style>): Estilo aplicado diretamente dentro do HTML.
CSS inline: Estilo aplicado diretamente em elementos específicos.

## 🎨 Funcionalidades
Organização em três colunas:
 🎥 Filmes e Séries
 🎵 Música e Podcasts
 🎮 Games e Tecnologia
 
Uso de imagens ilustrativas para cada seção.
Aplicação de três tipos de CSS (inline, interno e externo).
Estrutura semântica com <section>, <header> e <div>.
IDs e classes para facilitar a estilização.

## 📌 Observações

O projeto é educacional, voltado para prática de HTML e CSS.
